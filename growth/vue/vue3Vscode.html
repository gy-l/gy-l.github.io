vscode 插件安装和配置支持 vue3 #

一、常用插件 #

  1. 插件 Vue3 Snippets (用于 vue3 代码提示，语法高亮、智能感知、Emmet 等。替代 Vetur 插件)
  2. 插件 Volar
  3. 插件 ESLint
  4. 插件 Prettier
  5. 其他一些常用插件
  • Auto Rename Tag 修改 html 标签，自动帮你完成尾部闭合标签的同步修改
  • Auto Close Tag 自动闭合 HTML 标签
  • Vscode-icons 让 vscode 资源目录加上图标
  • Path Intellisense 自动路径补全、默认不带这个功能
  • Vue-color vue 语法高亮主题

二、常用插件配置 #

  1. 每次保存的时候自动格式化 搜索 format On Save，勾选前面的复选框

  2. 每次保存的时候将代码按 eslint 格式进行修复 搜索 Code Actions On Save，点击下方的在 setting 中编辑

	"editor.codeActionsOnSave": {
			"source.fixAll.eslint":true
		},

2
3

  1. 在函数名和后面的括号之间加个空格 搜索 Insert Space Before Function Parenthesis，并将前面的复选框勾上

  2. ESLint 添加 vue 支持 搜索 validate，选择左边的 ESLint，点击下方的在 setting 中编辑

	"eslint.validate": [
			"html","javascript","javascriptreact","vue"
		],

  1. 用单引号替代双引号 搜索 Single Quote，勾选前面的复选框

  2. 代码结尾以分号结束 搜索 Semi，去掉前面的复选框

  3. ESLint 插件修改(会暴露大量的空格问题警告) 解决方法就是：给.eslintrc.js 中的 rule 添加： 'prettier/prettier': 'off',（注意：该项会让前面 2 个设置失效）
