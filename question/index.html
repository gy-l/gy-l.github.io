vite press 部署 git 的方法 #

1、提交命令已写在deploy.sh文件里 2、可以build以后使用bash执行或者直接使用package.json里的scripts执行即可 3、如果链接github报错 433，可以修改修改host文件或者修改 git 代理地址。 更改代理方式如下：

git config --global http.proxy socks5://127.0.0.1:7890
git config --global http.https://github.com.proxy socks5://127.0.0.1:7890
# socks5查看自己的代理
